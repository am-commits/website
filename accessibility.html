Accessibility

In this site I have tried to follow web accessibility guidelines, namely the most recent (2.1) of the Web Content Accessibility Guidelines from W3C. There are text alternatives for content such as images, forms are labelled and there are multiple color scales for the data visualization. There has been an effort to make sure text remains readable in all cases, and is properly contrasted.
