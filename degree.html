Computer Science

Computer Science is, as is evident by the name, the study of computers. As opposed to engineering, Computer Science mainly deals with software and theories pertaining to software. It covers both practical parts, such as programming and implementation, and theoretical ones, such as design.

COM1001 Introduction to Software Engineering

COM1001 is the Software Engineering module. As such, it deals mainly with the creation of Software, especially the technical aspects of it such as designing, which includes gathering requirements, developing and testing. The module also includes group work, an important part of any software design process. A programming language is also introduced in it, Ruby, a general-purpose programming language that focuses on being simple.

COM1002 Foundations of Computer Science

COM1002 is the mathematics module. Its goal is to teach mathematical ideas that are useful within the scope of Computer Science, for example, mathematical logic and boolean algebra.

COM1003 Java Programming

Java is one of the most popular programming languages. It is a general-purpose programming language, meaning it has a wide variety of uses. This module deals with practically programming in Java.

COM1005 Machines and Intelligence

COM1005, as the name suggests, deals with AI, robots and computer intelligence in general. It introduces various theories regarding computer intelligence, and also deals with the MiRo robot , where the practical part of the module is introduced via implementing various algorithms and solutions. Other explored subjects include, but are not limited to, the history of AI, computer vision and human-robot interactions.

COM1006 Devices and Networks

COM1006 is more 'Computer-engineering' oriented. It deals with the interaction of various computer architectures and the logic behind it, as well as the connection of networks.

COM1008 Web and Internet Technology

COM1008's emphasis is on website creation, specifically on the development of an interactive, accessible and attractive page (much like this one) using HTML, CSS and JavaScript. Other than that, it also covers network security and safety.
